BaiduPanFilesTransfers
-----
#### 介绍使用
百度网盘批量转存工具,基于Python 3.8+Tkinter
详细介绍使用请访问:[小众软件](https://www.appinn.com/baidupan-files-transfers/)

#### 下载使用
点到release页面下载bpftUI.exe,直接打开使用

#### 自行打包
1. 克隆项目,在CMD中切换至项目目录
2. 使用pyinstaller打包:
``
pyinstaller -F -w -i bpftUI.ico bpftUI.py
``
